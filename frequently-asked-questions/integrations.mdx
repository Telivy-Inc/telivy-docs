<AccordionGroup>
  <Accordion title="Nodeware">
    1\. Go to Accounts > Integrations and enable Nodeware

    ![](/ScreenShot2025-01-09at1.22.24PM.png)2. Enter the Nodeware API key and save![](/ScreenShot2025-01-09at1.22.39PM.png)

    3\. Open the client in Nodeware and click on Manage Customer

    ![](/ScreenShot2025-01-09at1.25.43PM.png)

    4\. Copy the Customer ID shown in the sidebar

    ![](/ScreenShot2025-01-09at1.26.15PM.png)

    5\. On the Risk Assessment for the same client, enable Nodeware by visiting Setup > Optional Steps > Connect Nodeware and adding the customer ID.

    ![](/ScreenShot2025-01-09at1.23.19PM\(1\).png)

    6\. Once done, we will start fetching data and it should show up under Security > Network in 24 hours
  </Accordion>

  <Accordion title="Rewst">
    1. Set up a custom configuration - call it Telivy

    2. 2\. In Authentication Method, add

       *api-v1.telivy.com*

       as "Hostname" and choose

       *API Key*

    3. Navigate to the Telivy portal and look under Account >> Integrations to find your Telivy API key.

    4. In Test Configuration, set up /

       *api/v1/security/risk-assessments*

       as your Test Action URL Path

    5. Start configuring your workflow!
  </Accordion>
</AccordionGroup>
